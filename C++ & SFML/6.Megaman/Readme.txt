Objetivo:
Prueba de concepto de manejador de texturas, sprites animados y carga de spritesheet mediante un archivo .json; utilizando a Megaman como personaje.

Movimiento:
- Flechas izquierda y derecha para desplazamiento horizontal.
- Tecla A para saltar.
- Tecla B para disparar.
- Si se está quieto, se puede apretar flecha abajo + Tecla A para dezlizarse.
- Se puede cargar el arma de Megaman manteniendo la tecla B y soltando luego de un tiempo. Dependiendo de cuanto se mantenga, se lanzará 2 tipos de disparos cargados distintos.
